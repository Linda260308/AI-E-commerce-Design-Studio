# AI 电商海报 MVP 需求文档

**版本**: 1.0  
**日期**: 2026 年 3 月 18 日  
**作者**: Linda

---

## 一、产品愿景

帮助电商卖家和中小企业通过 AI 生成 + 可编辑文字，在 3 分钟内将普通产品照片转化为专业营销海报，解决现有工具"生成后无法修改文字"的痛点。

## 二、核心功能

### 2.1 AI 背景生成
- 使用 Nano Banana 2 或 Qwen-Image-2.0 模型
- 支持上传产品图，自动生成专业影棚级背景
- 支持文字提示词定制风格

### 2.2 可编辑文字
- **核心差异化功能**：生成后可修改文字内容
- 支持 10+ 字体样式
- 支持多种颜色选择
- 支持文字位置调整

### 2.3 背景去除
- 集成 rembg 自动抠图
- 支持手动微调
- 处理后的产品图无缝融入新背景

### 2.4 支付系统
- PayPal 集成
- 免费套餐：5 次生成（带水印）
- Pro 套餐：$9.9/月，50 次生成，高清无水印
- 年付套餐：$99/年（省 20%）

## 三、技术架构

### 3.1 前端 (Vercel 部署)
- **框架**: Next.js + React
- **样式**: Tailwind CSS
- **支付**: PayPal React SDK

### 3.2 后端 (AWS/GCP)
- **框架**: FastAPI (Python) 或 Node.js
- **数据库**: PostgreSQL (用户/订单)
- **缓存**: Redis (队列/缓存)
- **存储**: CloudFlare R2 (图片存储)

### 3.3 AI 服务
- **主模型**: Nano Banana 2 (通过 API 易)
- **备选**: Qwen-Image-2.0 (文字增强)
- **抠图**: 自部署 rembg

## 四、用户流程

1. 用户上传产品照片
2. AI 自动去除背景
3. 用户输入文字内容和风格偏好
4. AI 生成海报（含文字）
5. 用户可编辑文字（内容/字体/颜色）
6. 预览满意后下载
7. 免费用户带水印，Pro 用户高清无水印

## 五、MVP 范围

### 包含
- 基础上传和生成功能
- 3 种字体样式
- 5 种颜色选项
- PayPal 支付集成
- 用户账户系统

### 不包含 (V2)
- 批量生成
- 模板库
- 团队协作
- API 开放

## 六、成功指标

- 用户注册转化率 > 20%
- 免费转付费 > 5%
- 平均生成时间 < 30 秒
- 用户满意度 > 4.5/5

## 七、风险与挑战

1. **AI 生成质量**：需测试多个模型
2. **文字可编辑性**：技术难点，需验证可行性
3. **支付合规**：PayPal 商家账户审核
4. **成本控制**：AI API 调用成本
